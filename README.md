# Курсовой проект ONCHAT, предназначен для улучшения совместного времяприровождения за счет 
поиска единомышленников, будь то бизнес, игры, музыка, книги д что угодно люди сами решают что им интересно,
а за счет этих данных ему рекомендуются соответствующие чаты.
Благодаря сегригации каналов на публичные разделы, любой желающий сможет присоединиться к тому или иному чату
на его выбор.
